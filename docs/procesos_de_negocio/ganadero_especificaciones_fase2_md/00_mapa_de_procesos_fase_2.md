# GANADERO SaaS · Fase 2
## Mapa de procesos funcionales detallados

### Base de origen
Este paquete extiende el documento funcional base oficial del producto y desarrolla los procesos de fase 2 como especificaciones funcionales detalladas listas para seguir hacia matrices transversales, modelo funcional y posteriormente modelo de datos.

### Procesos incluidos
1. Vacunación
2. Tratamiento sanitario
3. Palpación o revisión reproductiva
4. Destete
5. Cambio de categoría
6. Descarte
7. Nacimiento

### Criterios de construcción aplicados
- no se reabre lo ya aprobado en el documento base
- se aterriza cada proceso a un nivel más útil para análisis funcional
- se fortalecen campos, validaciones, bloqueos, advertencias y trazabilidad
- se mantiene el principio de que el estado del animal se deriva de eventos
- se conserva la lógica de que Registrar es el punto formal de ejecución

### Estructura usada en cada documento
1. objetivo
2. alcance
3. roles que intervienen
4. disparador del proceso
5. precondiciones
6. datos de entrada
7. campos exactos
8. obligatorios y opcionales
9. valores por defecto
10. reglas de validación
11. bloqueos
12. advertencias
13. mensajes funcionales
14. flujo principal paso a paso
15. escenarios alternos
16. correcciones y anulaciones
17. resultado esperado
18. impacto en historial
19. impacto en estados derivados del animal
20. observaciones de diseño funcional

### Estado sugerido
- documentos listos para revisión funcional rápida
- aptos para pasar a capas transversales
- aptos para derivar matrices, casos de uso y modelo lógico
